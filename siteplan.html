Course work

During this Web Frontend Development block. This course builds on prior experience with Dynamic Web Fundamentals and programming. Students will focus on user experience, accessibility, compliance, performance optimization, and basic API usage. The course uses hands-on activities with students actually participating in simple web design and development projects. It is anticipated that students who complete this course will have firm grasp of the core web technologies of HTML, CSS, and JavaScript and be ready for the next sequence of application courses.
